# GitHub Pages Setup Guide

This guide explains how to set up GitHub Pages for the OpsToolkit project.

## Steps

1. Enable GitHub Pages in your repository settings.
2. Point DNS to GitHub Pages (A records to `185.199.108.153`, `185.199.109.153`, `185.199.110.153`, and `185.199.111.153` and a `CNAME` record to `<user>.github.io`).
3. Configure your custom domain in the GitHub Pages settings.
4. Wait for DNS propagation (this can take up to 48 hours).
5. Enable HTTPS in the GitHub Pages settings.
